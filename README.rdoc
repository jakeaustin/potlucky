== README

Potlucky is a platform for organizing who-brings-what to events with food.

Check out the app at jaustin-potlucky.herokuapp.com

Ruby v 2.1.2
Rails v 4.1.1
Configure with rake db:setup (seed data coming soon)

Contributing: (I'm flattered!) Fork and clone, submit pull request, and become the best of friends! I put this together relatively quickly so there are a lot of small (and a few big) improvements to be made, and I will be refactoring periodically when I find the time.
